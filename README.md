## ■ 概要

シンプルなカウンターアプリです。

基本機能に加え、拡張としてUI挙動の制御を実装しています。

---

## ■ 機能

- +1 / -1 カウント
- +10 / -10 カウント
- 最小値制限（0未満にならない）
- リセット機能
- 10の倍数でスタイル変化

---

## ■ 技術ポイント

### ■ 境界値制御（clamp）

```
if (next<0)next=0;
```

---

### ■ データ属性による制御

```
data-delta="10"
```

→ ボタン追加のみで機能拡張可能

---

### ■ UI制御（状態と見た目の分離）

```
if (value!==0&&value%10===0)
```

```
.is-multiple-of-10 {
  color:red;
}
```
